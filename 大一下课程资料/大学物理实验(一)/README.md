## BomLook注

这门课的课程标准因老师不同，实验报告和作业也是类似的，可能部分老师要求不同：有些老师可能要求做预习报告。

**此外，由于找不到原先的课程表，且考虑到每年的大物实验课程都会有所变动，所以在这里整理的大物实验报告，包括了大一下和大二上。**需要找大二上的大物实验时，直接到大学物理实验（一）下找即可。
