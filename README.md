## 🌟 共同寄语

这是前有古人，后希望亦有来者的一项工作，旨在更好地帮助大家能更方便、轻松地找到所需资料，更高质量地完成课业的学习，并延续软件领域开源的精神，共同进步。

初衷是减轻大家工作量，但是希望大家能保持追求卓越的学术作风、并且牢记学术道德。

”希望学弟学妹们能在我们的基础上，成为更加优秀的工程师、企业家和科学家。华南理工的旗帜将由你们扛起，你们都将成为发展和建设更加和谐美丽的粤港澳大湾区、中国、乃至全世界的中坚力量。“  – 计院开源前辈 YeewahChan
