Step8: Banking Project - Create a simple GUI program

Objective

In this exercise you will use random access file, GUI Components to manuplate data.

Directions

  1. Store and retrieve records relating to Customer (Does not consider Accounts of Customer temporarily) in project. Use the RandomAccessFile class. (You can reference "Case Study: Address Book")

  2. Program should allow the user to define the data schema of record and to creat new records; display records; delete records; sort records and search records.

  3. Viewing and manuplating records should be implemented by GUI application.
